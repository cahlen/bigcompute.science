# bigcompute.science

Open computational mathematics. Custom CUDA kernels. GPU clusters. AI-audited, not peer-reviewed.

**Human-readable. Agent-consumable.** Every result has structured metadata, JSON-LD, citation tags, and machine endpoints.

## Architecture

| Component | Location |
|-----------|----------|
| **Website** | This repo — Astro + KaTeX, Cloudflare Pages |
| **Experiments** | [cahlen/idontknow](https://github.com/cahlen/idontknow) — CUDA kernels, results, reviews, research agent |
| **MCP Server** | `workers/mcp/` — 23 tools, no auth, [mcp.bigcompute.science](https://mcp.bigcompute.science/mcp) |
| **Datasets** | [Hugging Face](https://huggingface.co/cahlen) — Zaremba, Kronecker, spectra, Ramanujan |

## Key Pages

| Page | Purpose |

## Machine Discoverability

Every finding page emits:
- **ScholarlyArticle** JSON-LD (Google Scholar)
- **Dataset** JSON-LD (Google Dataset Search) — when finding has a dataset
- **Highwire Press** meta tags (`citation_title`, `citation_author`, `citation_date`)
- **Dublin Core** metadata (`DC.title`, `DC.creator`, `DC.subject`)
- **IndexNow** pinged on every deploy (Bing, Yandex, etc.)

## Verification

All data-driven from `src/data/certifications.json` (generated by [sync_website.py](https://github.com/cahlen/idontknow/tree/main/scripts/reviews)):

- **Verification page**: dynamic cards with review ledger + issue tracker
- **Finding badges**: per-model verdicts, remediation status, commit links
- **Changelog**: auto-generated from git log

## Build

```bash
npm install
npm run build    # static output in dist/
npm run dev      # local dev at localhost:4321
```

Deployed via Cloudflare Pages — push to `main` auto-deploys.

## Stack

- [Astro](https://astro.build) + [KaTeX](https://katex.org) + [Cloudflare Pages](https://pages.cloudflare.com)
- MCP server: Cloudflare Worker (TypeScript)
- Data pipeline: `manifest.json` → `certifications.json` → `meta.json` → all pages
